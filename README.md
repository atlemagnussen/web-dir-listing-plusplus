# audio player

just a nodejs express server that should traverse a lib folder of audio files and stream them in a client

optimally organizing them in their folder structure

## run

### web client
```sh
cd web
npm run build
```

### server
create env-file `.env.prod`
```sh
cd server
npm run prod:ts
```

## debug

### web client
```sh
cd web
npm start
```
